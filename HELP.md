# TermFlow Help

## Global Keys
- **:**      - Open Command Palette
- **ESC**    - Close / Exit
- **f**      - Toggle Flow Mode
- **Enter**  - Add task
- **Space**  - Toggle task
- **Del**    - Remove task
- **q**      - Quit

## Flow Mode
- Single-task focus (read-only)
- Pomodoro auto-start
- Task auto-advance on completion
- Toggle visibility via Command Palette (:)

## Command Palette Commands
- Help: View Guide (External)
- Info: About (External)
- Theme: Dark/Light Mode
- Flow Mode Settings (while in Flow Mode)
